# 多语言翻译工具

在有自动化翻译工具之前，我们可以先使用 Copilot 进行手工翻译。借助 Copilot 的智能提示，可以快速生成翻译文本。

---

请将以下 C# 分析器的 Id 进行翻译。

| Id             | zh-hans  | zh-hant  | en                         |
|----------------|----------|----------|----------------------------|
| DL0000         | 未知错误。    | 未知錯誤。    | Unknown error.             |
| DL0000_Message | 发生了未知错误。 | 發生了未知錯誤。 | An unknown error occurred. |
